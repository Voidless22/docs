## Color Codes

`\ab = black`  
`\a-b = black (dark)`

`\ag = green`  
`\a-g = green (dark)`

`\am = magenta`  
`\a-m = magenta (dark)`

`\ao = orange`  
`\a-o = orange (dark)`

`\ap = purple`  
`\a-p = purple (dark)`

`\ar = red`  
`\a-r = red (dark)`

`\at = cyan`  
`\a-t = cyan (dark)`

`\au = blue`  
`\a-u = blue (dark)`

`\aw = white`  
`\a-w = white (dark)`

`\ay = yellow`  
`\a-y = yellow (dark)`

`\ax = previous color (if no previous \a? this would be the default mq2 color)`
